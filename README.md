# Zeal Auth
First attempt to create Zeal Authorization module

## Development using Gin Hot Reload feature
Refer to https://github.com/codegangsta/gin

Make sure gin is installed
```
go get github.com/codegangsta/gin
```
Go to Apps `src`
```
cd $GOPATH/src/github.com/ridzeal/zeal-auth
```
Run Server with Hot Reload
```
gin run main.go
```
